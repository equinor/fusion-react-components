### Installation

```sh
npm install @equinor/fusion-react-people
```

## Components

- [PeoplePicker](https://equinor.github.io/fusion-react-components/?path=/story/people-picker--basic) - Component for picking people from the people api.
- [PersonPicker](https://equinor.github.io/fusion-react-components/?path=/story/person-picker--basic) - Component for picking a single person from the people api.
- [PeopleViewer](https://equinor.github.io/fusion-react-components/?path=/story/people-viewer--basic) - Component to display list or tabel of selected people.
