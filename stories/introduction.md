## Packages

-   `@equinor/eds-core-react` is the core package for the EDS React components.
    
-   `@equinor/eds-icons` is a collection of svg data for the EDS components and can be used together with the `Icon` component in @equinor/eds-core-react. You can also use this data for populating your own svg elements.
    
-   `@equinor/eds-tokens` is a collection our EDS design tokens.
    
-   `@equinor/eds-data-grid-react` provides a data grid component that simplifies tabular data management while following EDS guidelines..
    
-   `@equinor/eds-lab-react` is an experimental package for testing and new upcoming EDS React components.
    
-   `@equinor/eds-utils` is our utility library for internal use in our core and lab packages.
