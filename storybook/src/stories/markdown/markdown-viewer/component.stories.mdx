# Markdown Viewer

## Properties/Attributes
| Name | Type | Default | Description
| ---- | ---- | ------- | -----------
| `value` | `string` | `''` | Markdown content

---

## Example Usage - Initial value as Child

```tsx
import { MarkdownViewer } from '@equinor/fusion-react-markdown';

<MarkdownViewer props>
  **some** markdown *text*
</MarkdownViewer>
```

## Example Usage - Initial value as Value
```jsx
import { MarkdownViewer } from '@equinor/fusion-react-markdown';

const markdown = "# My Heading";
<MarkdownViewer value={markdown} />
```
