# Components

## PersonAvatar

### Customizing

The avatar component has properties:

- `azureId` required unique Auzere ID of the person as `string`
- `size` size of the avatar as `AvatarSize` from `fwc-avatar`
- `clickable` enables ripple effect when avatar is clicked
- `disabled` disables the avatar
