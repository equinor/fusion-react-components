# PersonCard

## Properties/Attributes

In order for component to work, `azureId`, `upn` or `dataSource` is **required**.

Name                      | Type                            | Default                         | Description
------------------------- | -----------------------------   | -----------------------------   | -----------
`azureId`                 | `string`                        | `/`                             | Unique Azure ID for every person `required`
`upn`                     | `string`                        | `/`                             | Unique person email address.
`dataSource`              | `CardData**`                    | `/`                             | Custom data for avatar.
`size`                    | `PersonItemSize*`               | `medium`                        | Size of elements of the card.
`maxWidth`                | `number`                        | `300`                           | Maximum width of the component (px)
`contentHeight`           | `number`                        | `150`                           | Height of content (px)

---


\*  `PersonItemSize` is exported by `fwc-avatar`.
```ts
type PersonItemSize = 'small' | 'medium' | 'large';
```

## Example Usage

```tsx
import { PersonCard } from '@equinor/fusion-react-person';

// azureId
<PersonCard azureId="8a5f03ff-1875-4bf3-a3f4-aef1264e3bcc" />

// upn
<PersonCard upn="email@equinor.com" />

// dataSource
<PersonCard dataSource={{
        azureId: '8a5f03ff-1875-4bf3-a3f4-aef1264e3bcc',
        name: 'First Last Name (Bouvet ASA)',
        pictureSrc: 'https://i.imgur.com/GcZeeXX.jpeg',
        accountType: "Consultant",
        jobTitle: 'X-Bouvet ASA (PX)',
        department: 'FOIT CON PDP',
        mail: 'example@email.com',
        officeLocation: 'Stavanger',
        mobilePhone: '+47 999999999',
    }} />
```

## Person Provider

In order to use components such as PersonAvatar or PersonCard you need to have **PersonProvider** wrapper.

### Properties/Attributes

Name                      | Type                            | Description
------------------------- | -----------------------------   | -----------
`resolve`                 | `PersonResolver*`               | Person resolver for a provider. See [`fwc-person-provider`](https://github.com/equinor/fusion-web-components/tree/main/packages/person/src/person-provider) 

### Methods for PersonCard

Name                                    | Return Type                     | Description
--------------------------------------  | -----------------------------   | -----------------------------
`getCardDetailsByAzureId(azureId)`      | `CardData**`                    | Returns `CardData` for unique AuzureId.
`getCardDetailsByUpn(upn)`              | `CardData**`                    | Returns `CardData` for unique email address.


\*  `PersonResolver` is exported by `fwc-person-provider`.
```ts
interface PersonResolver {
    getPresence: (azureId: string) => Promise<PersonPresence> | PersonPresence;
    getDetails: (azureId: string) => Promise<PersonDetails> | PersonDetails;
    getImageByAzureId: (azureId: string) => Promise<AvatarData> | AvatarData;
    getImageByUpn: (upn: string) => Promise<AvatarData> | AvatarData;
    getCardDetailsByAzureId: (azureId: string) => Promise<CardData> | CardData;
    getCardDetailsByUpn: (upn: string) => Promise<CardData> | CardData;
    getListItemDetailsByAzureId: (azureId: string) => Promise<ListItemData> | ListItemData;
    getListItemDetailsByUpn: (upn: string) => Promise<ListItemData> | ListItemData;
}
```

\*\* `CardData` is exported by person avatar task.
```ts
type CardData = RequiredAndOptionalPick<
  PersonDetails,
  'name' | 'pictureSrc',
  'department' | 'jobTitle' | 'accountType' | 'mail' | 'mobilePhone' | 'positions' | 'manager'
>
```

`PersonPresence` is exported by `fwc-person-provider`.
```ts
type PersonPresence = {
    azureId: string;
    availability: PersonAvailability;
}

```
`PersonDetails` is exported by `fwc-person-provider`.
```ts
type PersonDetails = {
    azureId: string;
    name?: string;
    pictureSrc?: string;
    jobTitle?: string;
    department?: string;
    mail?: string;
    company?: string;
    mobilePhone?: string;
    accountType?: PersonAccountType;
    officeLocation?: string;
    positions?: Position[];
    manager?: Manager;
    managerAzureUniqueId?: string;
}

type Position = {
    id: string;
    name: string;
    project: {
        id: string;
        name: string;
    };
}

type Manager = {
    azureUniqueId: string;
    name?: string;
    pictureSrc?: string;
    department?: string;
    accountType?: PersonAccountType;
}
```
