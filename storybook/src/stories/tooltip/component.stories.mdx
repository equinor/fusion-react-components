### Description

This tooltip component is re exported [Tippy.js for React](https://github.com/atomiks/tippyjs-react) using the 
"out of the box" behavior called Default Tippy, with the built-in DOM rendering and the default CSS. 
>Tippy.js is the complete tooltip, popover, dropdown, and menu solution for the web, powered by Popper.

All of the native Tippy.js props can be passed to the component. In addition, there are 3 more props added 
specifically for the React component:
```className?: string```
```disabled?: boolean```
```visible?: boolean```

Visit [https://atomiks.github.io/tippyjs/v6/all-props/](https://atomiks.github.io/tippyjs/v6/all-props/) 
for detailed information on all the configurable Tippy.js properties.
